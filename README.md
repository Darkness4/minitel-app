## Features

- Authentication
- Apps
  - Sogo Mail
  - Campus EMSE
  - Prométhée
  - Logiciel Education
  - Gitlab EMSE
  - Annuaire
  - Imprimante
  - Wiki Minitel
- News
- Calendar
- Maps
- Report to Minitel
